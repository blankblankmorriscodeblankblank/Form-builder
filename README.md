# Form-builder
The component is built using standard JSX.

• It uses react-hook-form's register and handleSubmit avoiding any manual state tracking for the inputs.

• The form has all 10 required fields arranged in a clean, modern grid layout.

• Validation rules are properly configured for all fields (e.g., email patterns, minimum lengths, number constraints).

• Upon submission, it simulates a save operation (showing a loading state), displays a success message, and then uses reset() to clear all fields automatically.
